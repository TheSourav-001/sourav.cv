# Sourav Dipto Apu – CV Website

This repository contains the personal CV website of **Sourav Dipto Apu**, developed using **HTML, CSS, and JavaScript**.  
The website presents professional, academic, and personal information in a structured and user-friendly manner.

## Overview
The purpose of this project is to provide a digital version of the CV that is easily accessible, visually clear, and responsive across devices.  
The implementation is kept lightweight and framework-free to ensure simplicity and performance.

## Key Features
- Structured CV presentation
- Responsive layout for desktop and mobile devices
- Clean and professional user interface
- Subtle animations for better user experience
- Lightweight implementation with no external frameworks

## Technology Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
